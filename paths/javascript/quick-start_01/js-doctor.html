Doutores da Alegria

Utilizando estruturas de looping simples (do... while, while) crie um assistente médico que:

  • Pergunte como o paciente está se sentindo hoje (triste, alegre);
  • Enquanto ele estiver triste, conte uma piada para mudar isso;
  • Quando o paciente estiver alegre, agradeça e encerre;
